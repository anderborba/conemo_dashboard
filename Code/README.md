## Nota de escopo e precisão

- A existência de scripts auxiliares não implica, por si só, escopo ativo do MVP visual.
- O escopo visual vigente deve ser interpretado pelos documentos canônicos em `Docs/Plano-implementacao-dashboard.md` e `Docs/passo4-reconciliacao-mvp-visual.md`.
- Itens de backend/modelagem podem permanecer necessários para estrutura de dados e operação, mesmo quando não forem eixo principal da interface visual atual.
- Roadmap, backlog e materiais de apoio em código não equivalem a cronograma autorizado nem a autorização automática de execução de nova fase.
